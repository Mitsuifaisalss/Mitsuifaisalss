</div>

> **Gemma AI Chat** — *A fully local AI assistant with real-time web search*
> 
> 🏠 100% Local — `Ollama` + `Gemma 3` runs on your machine  
> 🔍 Smart Web Search — `DuckDuckGo` integration for live answers  
> ⚡ Streaming via `WebSocket` — instant token-by-token responses  
> 🌐 Public URL via `ngrok` — share with anyone  
> 🔐 JWT Auth + SQLite — secure user accounts  
> 📱 Mobile-responsive `React` + `Tailwind` UI  
>
