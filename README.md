## 🔥 Featured Projects

### 🤖 [Gemma AI Chat](https://github.com/Mitsuifaisalss/ai-chat-app)
> A fully local AI chat application powered by Gemma 3 with real-time web search

**What makes it special:**
- 🏠 **100% Local** — runs on your machine, zero data leaves your device
- 🔍 **Web Search** — DuckDuckGo integration for live, sourced answers
- ⚡ **Real-time Streaming** — WebSocket-powered instant responses
- 🌐 **Public Access** — ngrok tunnel for worldwide sharing
- 🔐 **User Authentication** — JWT-secured accounts with SQLite
- 📱 **Responsive Design** — works on mobile, tablet, and desktop

**Tech Stack:** `Python` `FastAPI` `React` `TypeScript` `Ollama` `WebSocket` `Tailwind CSS`

[🔗 Live Demo](https://lazily-unbundle-september.ngrok-free.dev) | [📂 Repository](https://github.com/Mitsuifaisalss/ai-chat-app)
